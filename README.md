# LlanquihueTourApp - Semana 6
## Clases creadas
- **`ServicioTuristico` (model/):** Clase base que define los atributos `nombre` y `duracionHoras`.
- **`RutaGastronomica` (model/):** Subclase que añade el atributo `numeroDeParadas`.
- **`PaseoLacustre` (model/):** Subclase que añade el atributo `tipoEmbarcacion`.
- **`ExcursionCultural` (model/):** Subclase que añade el atributo `lugarHistorico`.
- **`GestorServicios` (data/):** Clase encargada de crear instancias de prueba de los servicios.
- **`Main` (ui/):** Clase principal que ejecuta el programa y muestra los resultados en consola.

## Instrucciones para ejecutar Main

1.  Asegúrate de tener el proyecto `LlanquihueTourApp` abierto en tu IDE (por ejemplo, IntelliJ IDEA).
2.  Navega hasta la clase `ui.Main`.
3.  Haz clic derecho sobre el archivo `Main.java` y selecciona la opción **"Run 'Main.main()'"**.
4.  Observa la salida en la consola, donde se listarán todos los servicios turísticos creados con sus respectivos atributos.
